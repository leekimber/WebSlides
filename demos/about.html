About This Gallery

This gallery showcases presentations and articles built with WebSlides, an open-source framework for creating beautiful HTML presentations.

About WebSlides

WebSlides is a minimal HTML/CSS/JS framework for creating presentations and landings. Each <section> element represents a slide, and the framework handles navigation, transitions, and responsive behavior.

About This Fork

This is WebSlides Vanilla — a self-contained, offline-ready fork of WebSlides. All fonts, assets, and dependencies are local. No external CDN calls. Works offline.

Navigation

Desktop: Use arrow keys to navigate slides. Horizontal layout uses left/right arrows; vertical layout uses up/down arrows.

Mobile: Swipe up to go to the next slide, swipe down to go to the previous slide. Pinch to zoom out and see all slides at once.
