Leaflet Maps

Interactive maps in your WebSlides presentations.

This demo shows how to integrate Leaflet.js maps with WebSlides CSS.

Map with Context

Maps can be embedded alongside explanatory content.

Left Column

Use this space for descriptions, data, or narrative context about the map location.

Right Column

Add statistics, quotes, or related information that complements the visualization.

Multiple Locations

Plot multiple points with custom colored markers.

How to Add a Map

<!-- Include CSS -->
<link rel="stylesheet" href="static/css/webslides-leaflet.css">

<!-- Include JS -->
<script src="static/js/leaflet.js"></script>

<!-- Map container -->
<div id="map" style="height: 400px;"></div>

<!-- Initialize -->
<script>
  var map = L.map('map').setView([51.505, -0.09], 13);
  L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© OpenStreetMap'
  }).addTo(map);
</script>
